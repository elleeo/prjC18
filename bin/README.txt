Contiene i file bin delle versioni con la naming convention:
c18-HW-VERSIONE.bin
dove HW è l'hardware, valori possibili: hw1 (basetta iniziale senza display) hw2 (basetta con display e quattro bottoni)
VERSIONE è la versione software: 2.1 > v2_1

esempio:
c18-hw1-v2_0.bin è il firmware per hw 1 nella versione 2.0

il file VERSIONS-HW.txt contiene le versioni esistenti per l'hardware HW, la prima riga è l'ultima versione nella notazione con punti
